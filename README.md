## 🚀 Como rodar o projeto

Certifique-se de ter o **MAVEN** instalados no seu ambiente:

- macOS

```
brew install maven

mvn clean package
mvn exec:java -Dexec.mainClass="com.battleship.application.MainApplication"
```
